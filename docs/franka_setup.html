Robot setup

Franka FR3 setup notes

Status checked: 2026-07-10. Current main has partial Franka support, but not a complete hardware path.

Already present

  • rsc/robot/franka.urdf for FK and visualization.
  • src/validate/visualizer/franka.py for URDF smoke tests.
  • system/paradex2 and system/robothome include Franka IP entries.
  • paradex/transforms/coordinate.py and retargetor APIs mention franka.

Still missing

  • FrankaController is not on current main.
  • get_arm("franka") is still a stub.
  • Hand-eye capture supports xArm only.
  • Hand-eye FK is still hardcoded to xArm link6.

Recommended path

Port only Franka-specific files from origin/vlm_dex

The origin/vlm_dex branch contains useful Franka work, including paradex/io/robot_controller/franka_controller.py, src/capture/robot/franka_teaching.py, src/capture/robot/franka_replay_check.py, and src/capture/robot/franka_home.py. Do not merge the full branch without review; it diverges broadly from the current camera/process/site line.
